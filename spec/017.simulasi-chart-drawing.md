## 4. Perubahan Secara Logika (Logic Changes)

Secara garis besar, Recharts menggunakan format SVG yang sulit untuk dicoret-coret secara langsung tanpa plugin pihak ketiga yang spesifik. Pendekatan paling optimal dan ringan tanpa mengubah core Recharts adalah menerapkan **HTML5 Canvas Overlay**: sebuah elemen `<canvas>` transparan yang diposisikan tepat (`position: absolute`) menutupi area container Recharts.

**Logika Interaksi:**
1. **Mode Toolbar:** Buat state `drawMode` yang bisa berupa: `"none"` (default, chart interaktif dengan tooltip harga berjalan), `"draw"` (coret-coret freehand pencil), `"text"` (tambahkan teks), `"erase"` (hapus per baris/clear all).
2. **Freehand Drawing:** Menangkap event `onMouseDown`, `onMouseMove`, `onMouseUp`, dan `onMouseLeave` terpusat di atas canvas. Setiap pergerakan mouse dicatat sebagai list perintah path koordinat dan di-stroke secara real-time.
3. **Text Annotation:** Saat klik di canvas dengan mode "text", canvas memunculkan text prompt javascript (`prompt()`) atau memposisikan `<input type="text">` tersembunyi yang menerima ketikan, lalu di-render ke kanvas pakai `fillText()` pada koordinat klik mouse setelah user menekan respon OK/Enter.
4. **Z-Index Layering CSS:** Canvas harus selalu overlap di atas grafik SVG Recharts dan berada pada hierarki index teratas. Saat mode gambar dinon-aktifkan (`drawMode === "none"`), class styling root canvas disuntik state `pointer-events: none` agar pergerakan mouse menembus canvas, yang berarti Recharts dipaksa untuk dapat memproses tooltip tracking harga-nya tersendiri seakan tidak ada kanvas overlay menghalangi.

## 5. Perubahan Secara Code (Code Changes) & Kenapa Dilakukan

### Menambahkan Component Baru Canvas Overlay / Modifikasi Simulasi.tsx
Pilihannya dengan melakukan integrasi code `useRef` rendering canvas di dalam `Simulasi.tsx`.

- **State Management Canvas:**
  Menambahkan `drawMode` state (`'none'|'draw'|'text'`), lalu `isDrawing` (`boolean`).
- **Toolbar Overlay UI (Tombol Aksi Canvas):**
  Akan mendesain blok baris tombol Tool action (Icon/Teks Cursor, Pensil, Teks, Clear) menggunakan Absolute CSS position di Right/Top Pojok atas `.chart-wrapper` atau sebelum grafik.
- **Mengedit struktur JSX `.chart-wrapper`:**

```tsx
<div className="chart-wrapper" style={{ position: "relative" }}>
  {/* Tombol menu aksi diletakkan melayang di posisi absolute kanan atas */}
  <div className="drawing-toolbar"> 
       <button onClick={() => setDrawMode("none")}>🖱️</button>
       <button onClick={() => setDrawMode("draw")}>✏️</button>
       {/* Button text, clear */}
  </div>

  {/* Existing ResponsiveContainer Recharts ... */}

  {/* Layer Overlay Kanvas untuk digambari */}
  <canvas 
      className={`drawing-canvas ${drawMode !== "none" ? "active-draw" : "disabled-draw"}`}
      onMouseDown={startDrawing}
      onMouseMove={draw}
      onMouseUp={stopDrawing}
      onMouseLeave={stopDrawing} // Atasi drag kursor yang lolos dari batas kanvas (prevent stroke lari/bocor)
  />
</div>
```

### Mengapa Perubahan Code Ini Dilakukan?
- Meminimasi penggunaan dependensi ekstra luar (3rd party lib drawing coretan React) yang boros bandwith dan lambat dikelola karena sekedar interaksi "mencoret" bebas bisa dicapai murni melalui Standard Web API `CanvasRenderingContext2D`.
- Konteks interaktivitas default Recharts (tooltip responsif harga crypto live) akan tetap ada jika Pointer dikembalikan ke mode *None* dan pengguna puas mencoret/menulis di layernya.

## 6. Pseudocode Detail Integrasi DOM Mouse Handlers

```tsx
// 1. Deklarasi Hook Reference
const [drawMode, setDrawMode] = useState<"none" | "draw" | "text">("none");
const canvasRef = useRef<HTMLCanvasElement>(null);
const ctxRef = useRef<CanvasRenderingContext2D | null>(null);
const [isDrawing, setIsDrawing] = useState(false);

// 2. Setup Context dan Penyesuaian Dimensi Kanvas (useEffect Sync Resize)
useEffect(() => {
  if (!canvasRef.current) return;
  const canvas = canvasRef.current;
  
  // Mencocokkan resolusi nyata device dengan dimensi DOM (menghindari hasil gambar blurry)
  canvas.width = canvas.parentElement.clientWidth;
  canvas.height = canvas.parentElement.clientHeight;
  
  const ctx = canvas.getContext("2d");
  if(ctx) {
      ctx.lineCap = "round";
      ctx.lineJoin = "round";
      ctx.lineWidth = 2;
      ctx.strokeStyle = "#4ade80"; // Hijau/Warna aksen yang menarik visual
      ctx.font = "14px Inter, sans-serif";
      ctx.fillStyle = "#ffffff";
      ctxRef.current = ctx;
  }
}, [selectedCoin, drawMode]); // Sinkronisasi ulang kanvas jika coin ditukar / layout resize

// 3. Modulasi Event Tracking Mouse 
const startDrawingAction = (e: React.MouseEvent<HTMLCanvasElement>) => {
  if (drawMode === "none") return; 

  const { nativeEvent } = e;
  const ctx = ctxRef.current;
  const x = nativeEvent.offsetX;
  const y = nativeEvent.offsetY;

  if (drawMode === "draw" && ctx) {
      ctx.beginPath();
      ctx.moveTo(x, y);
      setIsDrawing(true);
  } else if (drawMode === "text" && ctx) {
      // Prompt Input Pop-up sederhana yang meminta pengguna memasukkan string
      const textToInject = prompt("Masukkan Teks Anotasi Grafik:");
      if (textToInject) {
          ctx.fillText(textToInject, x, y);
      }
      setDrawMode("none"); // Reset mode otomatis ke None agar tidak bocor ke pengetikan berturut-turut
  }
};

const drawAction = (e: React.MouseEvent<HTMLCanvasElement>) => {
  if (!isDrawing || drawMode !== "draw") return;
  const ctx = ctxRef.current;
  if(ctx){
      ctx.lineTo(e.nativeEvent.offsetX, e.nativeEvent.offsetY);
      ctx.stroke();
  }
};

const stopDrawingAction = () => {
  if (isDrawing && ctxRef.current) {
      ctxRef.current.closePath();
  }
  setIsDrawing(false);
};

// 4. Reset Button Action
const clearCanvasAction = () => {
   const canvas = canvasRef.current;
   const ctx = ctxRef.current;
   if(canvas && ctx) {
       ctx.clearRect(0, 0, canvas.width, canvas.height); // Blank out slate pixel
   }
}
```

## 7. Tracing Testcase (Simulasi Alur User)

1. **User Memilih Koin:** User masuk ke `Tab Simulasi`, memilih tipe aset `Ethereum (ETH)`. Grafik SVG Recharts merender lekukan garis harga Ethereum.
2. **User Ingin Coret Chart:** User melihat ada Toolbar Transparan yang mengambang di pojok kanan grafik dengan icon tools berupa `🖱️(Pointer)`, `✏️(Draw)`, `🔠(Text)`, `🗑️(Clear)`.
3. **Memicu Tombol Gambar (✏️ Draw):**
   - User meng-klik `✏️`.
   - Node status `drawMode` di set nilainya menjadi `"draw"`.
   - Element DOM `<canvas>` mendapat mutasi class yang merubah property CSS `pointer-events` menjadi `auto`.
4. **Coretan Bebas Menggaris di Atas Grafik:** Saat mouseDown ditahan dan digeser-rentang (drag), titik pointer cursor mouse meninggalkan garis tinta tebal warna hijau konstan di atas bentangan layar canvas/grafik.
5. **Cek Respons Tooltip Waktu & Harga:** Selama User *masih* berada di mode pen (mencoret), Tooltip Harga custom bawaan dari `Recharts` kuncian tidak merespon/hilang tertutup lapisan. Ini adalah efek limitasi *pointer trap*.
6. **Melihat Titik Harga Lanjutan:** User kini menekan opsi *🖱️ Pointer Kursor* (`"none"`). Mode interaksi gambar disable, class CSS Canvas dimutasi jadi `pointer-events: none`. Jika user meletakkan kursor mouse mengambang di atas chart grafis, jejak **garis coretan/pen sebelumnya TETAP utuh terlihat** meng-overlay layout-nya, namun sistem Tooltip Harga dari Recharts menembus mendeteksi hover dan kembali merefleksikan harga live mengikuti titik sumbu X secara presisi tanpa nge-bug.
7. **Injeksi Teks Label:** User klik tool *🔠 Text*. Status memicu mode text. User mengklik pucuk/puncak dari lekukan grafik. Terbuka browser prop prompt JS. User mengetik "Rebound titik X", dan teks tercetak pas di x-y tempat klik.
8. **Pembersihan Logikal:** User menekan tombol *🗑️ Clear*. `clearRect` dipanggil pada kanvas, total isi coretan menguap lenyap namun grafik trading utama sama sekali tak hilang/pudar.

   - Taruh kursor di atas coretan tersebut. Pastikan *Tooltip Info Popup Recharts* berjalan menyala lagi dari layer bawah, menampilkan koordinat chart dibaliknya tanpa menutupi coretan Anda yg sudah ditarik.
6. **Uji Pembubuhan Teks (Typography Stamp):**
   - Pilih tool **Label Teks**. 
   - Klik di sembarang pixel area, pastikan Prompt input OS browser muncul, isi string tulisan. Tutup prompt dan pastikan string huruf terefleksi pada posisi pixel itu bersiap mendanai titik support/resistance pointnya.
7. **Uji Fitur Kanvas Clean-up (Clear Event State):**
   - Klik tool **Tong Sampah**.
   - Pastikan canvas menjadi bersih tanpa coretan tinta/huruf satu pointpun. Cek Recharts Chart masih ber-animasi/hidup.
8. **Uji Validasi Leak Bocor Rendering Re-size (Krusial Bounds Testing):**
   - Tarik tools Pensil garis panjang melewati batas window parent div layout yang dirender (Lalui area padding batas layout luarnya dan trigger event `onMouseLeave`).
   - Lepas diluar bounds, lalu retur kembalikan pointer ke tengah area chart. Kondisi fatal biasa menyisakan Path terus mengeksekusi tarikan stroke menggantung (bug mouseUp di luar trigger event). Pastikan *`onMouseLeave` fallback trigger event handler stopDrawing* menangkap ini dan mencegah anomali sirkuit coretan macet lengket.
