## 3. Codebase Analysis (Kondisi Saat Ini)

- **`Dashboard.tsx` (Line 425-495):** Menampilkan seluruh `portfolioWithPrice` tanpa limitasi (`portfolioWithPrice.map(...)`). Jika user punya banyak coin, layar akan scroll terus ke bawah. Ini membuat desain terlihat kurang rapi. Tidak ada fitur / tombol "Lihat Semua" untuk portofolio.
- **`Dashboard.tsx` (Line 498-540):** Bagian Pasar Kripto menggunakan `market.slice(0, 7)`. Desainnya saat ini standar *border-box* tanpa kedalaman visual (depth).
- **`Dashboard.css`:** Sudah menggunakan tema gelap dengan sedikit *transparency*, namun masih kurang berasa *premium*. Hover effect hanya geser `translateY(-2px)` dengan shadow hitam biasa. Peringatan profit ($) dan persen (%) kadang tidak terlalu menyala.

## 6. Tracing Scenarios

1. **User punya 1 atau 2 koin:**
   - Di Dashboard, muncul 2 koin tersebut secara utuh.
   - Tombol "Lihat Semua →" **TIDAK** muncul karena `<= 3`. Tidak perlu pop-up.
2. **User punya 5 koin:**
   - Di Dashboard, muncul hanya 3 koin teratas (koin yang mungkin paling lama/baru dibeli).
   - Tombol "Lihat Semua →" muncul.
   - Klik "Lihat Semua" -> Pop-up modal muncul dari bawah (animasi `slideUp`).
   - Modal berisi ke-5 koin tersebut dengan urutan, menampilkan logo, harga aset satuan `currentPrice`, total kuantitas, dan persentase Profit/Loss.
   - Klik di luar modal / tombol silang (x) -> Pop-up menutup.
