## 2. Tujuan yang Dipahami

Me-redesign halaman **Chat** (`/chat/:conversationId`) agar tampilan dan perilakunya mirip seperti **WhatsApp** — khususnya dari sisi **user** yang chat dengan **mentor** via halaman Mentor (`Ai.tsx`).

### User Flow Saat Ini:
```
Dashboard.tsx → Tab "Mentor" → Ai.tsx → Klik "Chat Sekarang"
→ POST /api/chat/conversation → navigate(/chat/:conversationId)
→ Chat.tsx (shared page, dipakai user & mentor)
```

#### [MODIFY] `Backend/backend/routes/chatRoutes.js`

**Kenapa:** Perlu endpoint untuk bulk-mark unread messages saat user buka chat. Tanpa ini, user harus klik satu-satu pesan.

**Perubahan — tambah endpoint baru:**
```javascript
// 5️⃣ MARK ALL MESSAGES AS READ
router.put("/mark-read/:conversationId", auth, async (req, res) => {
  try {
    const userId = req.user.id;
    const result = await Message.updateMany(
      {
        conversationId: req.params.conversationId,
        receiver: userId,
        isRead: false
      },
      { isRead: true }
    );

    res.json({ success: true, modifiedCount: result.modifiedCount });
  } catch (error) {
    console.error("MARK READ ERROR:", error.message);
    res.status(500).json({ success: false });
  }
});
```
