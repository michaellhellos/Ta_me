## 2. Tujuan yang Dipahami

Dua fitur utama di halaman **Belajar** (`Belajar.tsx`):

### 2.1 Tampilkan XP per Quiz di List Materi
- Di halaman daftar materi/quiz, setiap quiz card menampilkan **berapa XP yang sudah dikumpulkan** user dari quiz tersebut
- Contoh: quiz card menampilkan "8/10 XP" jika user sudah mengerjakan dan mendapat skor 8

### 2.2 Tambah Tab "Jadwal Mentoring"
- Halaman Belajar dibagi jadi **2 tab**: **Quiz** dan **Jadwal Mentoring**
- Tab Jadwal Mentoring menampilkan upcoming mentoring schedules (yang di-upload oleh mentor)
- Data dari API `GET /api/schedule/upcoming` yang sudah tersedia

## 6. Logic / Pseudocode

### 6.1 Fetch User XP Scores

```ts
interface NilaiRecord {
  materiId: string;
  score: number;
  totalSoal: number;
}

const [nilaiMap, setNilaiMap] = useState<Record<string, NilaiRecord>>({});

useEffect(() => {
  const userData = localStorage.getItem("user");
  if (!userData) return;
  const user = JSON.parse(userData);

  fetch(`http://localhost:5000/api/nilai/user/${user._id}`)
    .then(res => res.json())
    .then(data => {
      if (data.success) {
        const map: Record<string, NilaiRecord> = {};
        data.data.forEach((n: NilaiRecord) => {
          map[n.materiId] = n;
        });
        setNilaiMap(map);
      }
    })
    .catch(err => console.error(err));
}, []);
```
