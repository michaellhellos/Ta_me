
## 2. Tujuan yang Dipahami

Menambahkan **tampilan XP points** tepat di atas section **"📝 Quiz Tersedia"** pada halaman Belajar (tab Quiz).

Secara spesifik:
- Menampilkan total XP yang sudah dikumpulkan user dari quiz yang telah dikerjakan
- Posisi: **langsung di atas** judul "📝 Quiz Tersedia" (bukan di atas seluruh page, itu sudah ada)
- Memberikan konteks berapa XP yang dimiliki user sebelum memilih quiz selanjutnya

### 2.1 Kenapa Tujuan Ini

Saat ini, halaman Belajar sudah memiliki XP summary banner di **paling atas** quiz tab (line 341-351 `Belajar.tsx`). Namun banner ini:
1. **Hanya muncul jika `totalEarnedXp > 0`** — user baru tidak melihat apa-apa
2. **Posisinya jauh dari "Quiz Tersedia"** — kalau user scroll ke bawah (melewati "Quiz Selesai" section), banner hilang dari viewport
3. **User ingin konteks XP tepat di area keputusan** — di atas daftar quiz yang tersedia untuk dikerjakan, agar user tahu berapa XP yang dimiliki sebelum mengerjakan quiz baru

### Scenario 4: User baru mengerjakan quiz lalu kembali ke list

1. User klik quiz "Materi DEF" → jawab → submit → score 6 XP
2. `nilaiMap` updated in-place: `{ "m1": { score: 8 }, "m2": { score: 6 }, "m3": { score: 10 } }`
3. User klik "Kembali ke Daftar Quiz"
4. React re-render:
   - `totalEarnedXp` = 24 (8+6+10)
   - `completedMateri.length` = 3
   - `pendingMateri.length` = 1
5. XP point card langsung menampilkan: **⭐ 24 XP | 3 selesai | 1 tersisa**
