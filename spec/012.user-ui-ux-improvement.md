## 2. Tujuan yang Dipahami

Memperbaiki dan memperbagus **seluruh halaman user** dari sisi UI/UX. Halaman user meliputi semua yang diakses dari **`/dashboard`**, yaitu 5 tab:

1. **Beranda** — `Dashboard.tsx` *(sudah di-redesign di spec 007)*
2. **Simulasi** — `Simulasi.tsx` — trading + chart + riwayat
3. **Belajar** — `Belajar.tsx` — materi edukasi + quiz
4. **Komunitas** — `Komunitas.tsx` — leaderboard + forum diskusi
5. **Mentor** — `Ai.tsx` — mentor list + jadwal mentoring

Plus halaman pendukung:
6. **Chat** — `Chat.tsx` — real-time chat dengan mentor
7. **Onboarding** — `Onboarding.tsx` *(sudah di-implementasi di spec 011)*

**Fokus utama:** Sub-page Simulasi, Belajar, Komunitas, Mentor, dan Chat — karena Beranda dan Onboarding sudah di-improve di spec sebelumnya.

## 6. Logic / Pseudocode

### 6.1 Toast Component

```tsx
// Toast.tsx
interface ToastProps {
  message: string;
  type: "success" | "error" | "info";
  onClose: () => void;
}

const Toast = ({ message, type, onClose }: ToastProps) => {
  useEffect(() => {
    const timer = setTimeout(onClose, 3000);
    return () => clearTimeout(timer);
  }, [onClose]);

  const icons = {
    success: <CheckCircle />,
    error: <XCircle />,
    info: <Info />,
  };

  return (
    <div className={`toast toast-${type}`}>
      {icons[type]}
      <span>{message}</span>
      <button onClick={onClose}><X size={14} /></button>
    </div>
  );
};
```

### 6.2 Simulasi — Custom Tooltip

```tsx
// Custom tooltip untuk recharts
const CustomTooltip = ({ active, payload, label }: any) => {
  if (!active || !payload?.length) return null;
  return (
    <div className="custom-tooltip">
      <span className="tooltip-time">{label}</span>
      <strong className="tooltip-price">
        ${Number(payload[0].value).toLocaleString()}
      </strong>
    </div>
  );
};

// Penggunaan:
<Tooltip content={<CustomTooltip />} />
```

### 6.6 Komunitas — Avatar Initial Letter

```tsx
// Reuse function dari Dashboard/Ai:
const avatarGradients = [
  "linear-gradient(135deg, #6366f1, #8b5cf6)",
  "linear-gradient(135deg, #06b6d4, #38bdf8)",
  "linear-gradient(135deg, #22c55e, #14b8a6)",
  "linear-gradient(135deg, #f59e0b, #f97316)",
  "linear-gradient(135deg, #ec4899, #f43f5e)",
];

const getAvatarGradient = (name: string) => {
  let hash = 0;
  for (let i = 0; i < name.length; i++) {
    hash = name.charCodeAt(i) + ((hash << 5) - hash);
  }
  return avatarGradients[Math.abs(hash) % avatarGradients.length];
};

// Post avatar JSX (replace emoji):
<div 
  className="post-avatar"
  style={{ background: getAvatarGradient(post.authorName) }}
>
  {post.authorName.charAt(0).toUpperCase()}
</div>
```

### 6.7 Chat — Auto-scroll & Partner Name

```tsx
// Auto-scroll ref
const messagesEndRef = useRef<HTMLDivElement>(null);

useEffect(() => {
  messagesEndRef.current?.scrollIntoView({ behavior: "smooth" });
}, [messages]);

// Partner name — dari conversation data
const [partnerName, setPartnerName] = useState("Chat Room");

useEffect(() => {
  if (!conversationId) return;
  
  axios.get(`http://localhost:5000/api/chat/conversation/${conversationId}`, {
    headers: { Authorization: `Bearer ${token}` }
  }).then(res => {
    const conv = res.data.data;
    // Find the other participant
    const partner = conv.participants?.find((p: any) => p._id !== user._id);
    if (partner) setPartnerName(partner.name);
  }).catch(() => {});
}, [conversationId]);

// Message timestamp
const formatTime = (msg: Message) => {
  if (!msg.createdAt) return "";
  return new Date(msg.createdAt).toLocaleTimeString("id-ID", {
    hour: "2-digit",
    minute: "2-digit",
  });
};

// JSX:
<div className="msg-bubble">
  <p>{msg.text}</p>
  <span className="msg-time">{formatTime(msg)}</span>
</div>

// Auto-scroll anchor:
<div ref={messagesEndRef} />

// Empty state:
{messages.length === 0 && (
  <div className="chat-empty">
    <MessageCircle size={48} />
    <p>Mulai percakapan!</p>
    <small>Kirim pesan pertamamu 💬</small>
  </div>
)}
```
