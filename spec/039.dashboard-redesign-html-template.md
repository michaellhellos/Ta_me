## 1. Tujuan yang Diinginkan

Merombak total tata letak (layout) dan gaya visual halaman `Dashboard.tsx` agar menyerupai referensi struktur HTML yang diberikan user, namun tetap **menyesuaikan dengan design system murni (Vanilla CSS) yang sudah eksis di `global.css`**.

**Kenapa tujuan ini penting?**
- Desain sebelumnya (vertical stacking) terasa "bootstrapped" dan memakan ruang ke bawah.
- Desain baru mengusung gaya *Bento Grid*, *Horizontal Scrolling* untuk portofolio, dan pembagian porsi layar 2/3 (Market Table) vs 1/3 (News Intel) sehingga memaksimalkan area *viewport* desktop.
- Mengintegrasikan estetika *Glassmorphism* & *Avant-Garde* pada level struktural yang jauh lebih rapi.

## 2. Codebase Analysis & Integrasi Desain

### 2.1 Kondisi Saat Ini vs Target
- **Aplikasi Eksisting:** Menggunakan modul CSS murni (`Dashboard.css` dan `global.css`) tanpa setup framework Tailwindcss v4 yang berjalan (hanya ada package, tidak di-define pada config/vite plugin). 
- **Referensi HTML:** Menggunakan Tailwind CSS CDN murni dengan *custom color palette*.
- **Integrasi (Deep Logic):** 
  Sesuai dengan *Directive: Intentional Minimalism*, kita **TIDAK AKAN** melempar script Tailwind CDN ke dalam `index.html` demi satu halaman, karena berpotensi merusak *global styles* React yang sudah ada dan memicu *FOUC (Flash of Unstyled Content)*. 
  Sebagai gantinya, saya akan **menerjemahkan struktur HTML Tailwind tersebut ke dalam sintaks React (`className`) dan men-styling-nya secara native di `Dashboard.css`** menggunakan variabel warna `global.css` (misal `var(--surface-1)`, `var(--primary)`, dll).

### 2.2 Pemetaan Komponen Utama

| Ref HTML | React Component Equivalent | Aksi CSS |
|----------|----------------------------|----------|
| Hero Section (Total Balance) | `total-asset` section (`Dashboard.tsx`) | Ubah menjadi layout lebar dengan tata letak teks raksasa di kiri & tombol aksi di kanan/bawah hero. Tambahkan ornamen SVG *curve* absolut di kanan. |
| Portfolio Bento (Horizontal) | `portfolio-card` list | Buat container *horizontal scroll* (`display: flex; gap: 1rem; overflow-x: auto`). Kartu portofolio dipadatkan menjadi *Bento Box* kotak bersudut tumpul tebal. |

2. **Hero Section (`total-hero`):**
