### 3.3 Data Flow (Current)

```
Dashboard mounts → fetchData():
  1. GET /api/trade/coins → CoinGecko market data (id, name, symbol, current_price, image, price_change_percentage_24h)
  2. GET /api/user/me → { id, name, email, balance, role }
  3. GET /api/trade/history → Transaction[] (type, coinId, name, price, quantity, total)

Portfolio computation (client-side):
  • Group BUY transactions by coinId
  • Subtract SELL transactions by coinId
  • Calculate weighted avg buy price from totalCost / totalBought
  • Get current price, symbol, image from market data match
  • Compute: totalValue, invested, profitDollar, profitPercent per coin
  • Compute: cryptoValue, totalCost, totalProfitDollar, totalProfitPercent, totalAsset (sums)
```

## 7. Logic / Pseudocode

### 7.1 formatPrice Helper

```typescript
/**
 * Format harga dengan decimal places yang sesuai berdasarkan magnitude:
 * - >= $1     → 2 decimal  (contoh: "$98,567.00", "$1.23")
 * - >= $0.01  → 4 decimal  (contoh: "$0.1234")
 * - < $0.01   → 6 decimal  (contoh: "$0.000123")
 */
const formatPrice = (price: number): string => {
  if (price >= 1) {
    return price.toLocaleString(undefined, { 
      minimumFractionDigits: 2, maximumFractionDigits: 2 
    });
  } else if (price >= 0.01) {
    return price.toLocaleString(undefined, { 
      minimumFractionDigits: 4, maximumFractionDigits: 4 
    });
  } else {
    return price.toLocaleString(undefined, { 
      minimumFractionDigits: 6, maximumFractionDigits: 6 
    });
  }
};
```

### 7.2 Market Cap Formatting

```typescript
// CoinGecko market data sudah return market_cap sebagai number
// Contoh: bitcoin market_cap = 1.92e12 (triliun)
// Format: "> $1T → $X.XT", "> $1B → $X.XB", "> $1M → $X.XM"

// Inline di JSX:
{coin.market_cap && (
  <span className="market-cap-label">
    MCap: ${coin.market_cap >= 1e12
      ? `${(coin.market_cap / 1e12).toFixed(1)}T`
      : coin.market_cap >= 1e9
        ? `${(coin.market_cap / 1e9).toFixed(1)}B`
        : `${(coin.market_cap / 1e6).toFixed(1)}M`}
  </span>
)}
```
